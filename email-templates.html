📧 Trackly Email Templates

Professional, beautiful email templates for Firebase Authentication

📋 How to Use These Templates

  1. Click the "Copy HTML" button on any template below
  2. Go to your Firebase Console → Authentication → Templates
  3. Select the email type (Email verification, Password reset, etc.)
  4. Click "Edit template"
  5. Paste the HTML code into the template editor
  6. Save and test!
