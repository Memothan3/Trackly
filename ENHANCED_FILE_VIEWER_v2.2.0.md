### 🧠 **Smart File Naming System**
- **Context-Aware Names**: Uses transaction data for meaningful filenames
- **Format**: `YYYY-MM-DD_type_amount_merchant_category_filetype_HHMMSS.ext`
- **Example**: `2026-04-15_expense_25_coffee_shop_food_receipt_143022.jpg`
- **Deduplication**: Timestamp ensures unique filenames
- **Metadata Storage**: Preserves original filename and transaction context
